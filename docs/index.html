⚡ CAFE-u

An AI Agent that monitors how users interact with your web app. When it detects frustration (rage clicks, dead clicks, hesitation), it automatically adapts the UI to resolve the issue — in under 500ms.

Open-source · Self-hosted · MIT licensed
Built with scikit-learn · FAISS · DeepSeek LLM · FastAPI · 10KB JS agent

🧠 How It Works

📡
1. Signal
Browser detects rage click, dead click, hesitation
→
📊
2. ML Classifier
Logistic Regression predicts frustration probability (0-1)
→
🔍
3. RAG Retrieval
FAISS vector search finds top-3 similar past cases
→
🤖
4. AI Agent
LLM (DeepSeek) decides which adaptation tool to use
→
✅
5. Adaptation
Agent applies DOM fix — highlight, tooltip, sticky CTA
🧮

ML Classifier

13 features per signal (type, frequency, duration, density). LogisticRegression trained online from user feedback.

scikit-learn
🔎

RAG Memory

FAISS vector index stores past signal→adaptation pairs. Retrieves top-3 most similar cases for context-aware decisions.

FAISS
🤖

LLM Agent

7 adaptation tools available. DeepSeek V4 Flash decides tool + params. Falls back to rule-based heuristic when offline.

DeepSeek
🌐

Browser Agent

10KB JS snippet. Zero dependencies. Detects 6 signal types, applies 7 adaptation actions. Runs standalone or with engine.

10KB JS

📦 Get Started in 2 Lines

Web — Add to any HTML page
<!-- No build step. Just a script tag. -->
<script src="https://cdn.jsdelivr.net/npm/cafeu-agent@0.1/dist/cafeu.min.js"
        data-key="my-app"></script>
Engine — Full AI Pipeline (Docker)
# One command. ML + RAG + LLM.
docker run -p 8080:8080 \
  ghcr.io/akhichalasani/cafeu-engine
